# Project: Alpha-Sam

## 목적

Alpha-Sam은 사용자의 자산(주식, 코인, 기타 투자 대상)의 **현재 평가액과 수익률을 한눈에 보여주는 웹 대시보드**를 목표로 합니다.  
사용자는 여러 자산을 등록하고, 각 자산의 매수 정보와 현재 시세를 기반으로 전체 포트폴리오의 성과를 직관적으로 확인할 수 있습니다.

## 주요 기능

1. 자산 관리
   - 종목(티커, 이름) 등록/수정/삭제
   - 매수 단가, 수량, 매수 일자 관리
2. 시세 조회
   - 외부 시세 API를 통해 현재 가격 조회
   - 주기적 업데이트 또는 사용자의 수동 새로고침
3. 수익률/평가액 계산
   - 자산별 평가액, 손익, 수익률 계산
   - 전체 포트폴리오 단위의 총 평가액/총 수익률 집계
4. 대시보드 UI
   - 테이블 및 카드 형태의 자산 목록
   - 포트폴리오 요약(총 자산, 총 손익)
   - 간단한 시각화(예: 자산 비중, 수익률 분포 등)

## 대상 사용자

- 개인 투자자(주식/코인/펀드 등)를 여러 개 보유하고 있는 사람
- 자산을 엑셀이나 수기로 관리하던 사용자를 웹 기반으로 옮겨주고 싶은 경우

## 비기능 요구사항 (간단)

- 빠른 로딩 속도 (가벼운 UI, 최소한의 네트워크 왕복)
- 모바일 브라우저에서도 사용 가능한 반응형 레이아웃
- 민감 정보(계정, 자산 데이터)는 안전하게 저장 및 전송
