# Handovers: To QA Tester

## 날짜
2025-12-09

## 현재 상황 (Context)
- 백엔드에 사용자 인증(User Model, JWT Auth API) 기능이 구현되었습니다.
- 프론트엔드에 로그인/회원가입 페이지와 Navbar, Footer 업데이트가 완료되었습니다.
- 이제 실제 회원가입부터 로그인, 로그아웃 흐름이 정상 작동하는지 검증해야 합니다.

## 해야 할 일 (Tasks)
1. **회원가입 테스트 (`/signup`)**:
   - 신규 이메일/비밀번호 입력 시 정상 가입 및 로그인 페이지 이동(또는 자동 로그인) 확인.
   - 이미 존재하는 이메일 입력 시 에러 메시지 표시 확인.
2. **로그인 테스트 (`/login`)**:
   - 올바른 계정 입력 시 Dashboard(`/`)로 이동 및 Navbar 상태 변경(Logout 버튼 표시) 확인.
   - 틀린 비밀번호 입력 시 에러 처리 확인.
3. **UI 요소 검증**:
   - 모든 페이지 하단에 **Footer**가 잘 표시되는지 확인.
   - Navbar의 반응형(햄버거 메뉴) 동작 재확인.
   - 로그아웃 버튼 클릭 시 로그인 상태가 풀리고 메인/로그인 페이지로 이동하는지 확인.
4. **API 보안 테스트 (Swagger UI)**:
   - `POST /api/v1/auth/signup` 및 `login` 직접 호출 테스트.
   - `GET /api/v1/users/me` 호출 시 토큰 없이 접근하면 401 Unauthorized 뜨는지 확인.

- 인증 기능 완전 동작 확진.
