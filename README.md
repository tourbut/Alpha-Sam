# Alpha-Sam

Alpha-Sam is an AI-powered Asset Management Dashboard built with FastAPI and SvelteKit.

## Project Structure

- `backend/`: FastAPI application (Python)
- `frontend/`: SvelteKit application (TypeScript)

## Getting Started

### Backend

```bash
cd backend
# Install dependencies
pip install -r requirements.txt
# Run server
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
# Install dependencies
npm install
# Run development server
npm run dev
```
