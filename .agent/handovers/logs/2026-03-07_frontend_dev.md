# Handovers: To Frontend Developer

## 날짜
- 2026-03-07

## 브랜치
- `feature/frontend-pdf-upload`

## 현재 상황
- 포트폴리오 관리를 자동화하기 위해, 사용자가 토스증권 거래내역서(PDF)를 업로드하면 백엔드에서 거래내역을 파싱해 DB에 저장하는 파일업로드 API가 개발될 예정입니다.
- 백엔드 작업과 연계하여 프론트엔드에 사용자가 파일을 업로드할 수 있는 UI 구현이 필요합니다.

## 해야 할 일
1. 포트폴리오 목록 페이지나 생성 모달에 "토스증권 거래내역 업로드"를 위한 UI(버튼 및 파일 선택 폼)를 추가하세요.
2. 백엔드 파일업로드 엔드포인트(`POST /api/v1/portfolios/upload/toss` 등)로 사용자가 선택한 파일을 `FormData` 형태로 전송하는 API 연동 로직을 작성하세요.
3. 파일 업로드 중 표시될 로딩 상태(스피너 등), 성공 및 에러 모달/피드백을 적절히 표시하세요.
4. 등록이 완료된 후 사용자의 포트폴리오 목록 데이터 갱신을 트리거하세요.

## 기대 산출물
- 사용자가 PDF 파일을 업로드할 수 있는 Svelte 컴포넌트 구현
- 백엔드 업로드 API 호출 연동 및 에러 처리
- 파일 업로드 성공 후 UI 새로고침 기능

## 참고 자료
- 백엔드 개발에 따른 API Spec (Swagger 문서 참고)
