# Alpha-Sam 서비스 및 기능 사용 설명서

## 1. 서비스 소개 (Service Overview)

**Alpha-Sam**은 개인 투자자를 위한 **직관적인 자산 관리 및 포트폴리오 트래킹 서비스**입니다.
복잡한 투자 내역을 한곳에서 관리하고, 실시간 시세와 연동하여 정확한 수익률을 분석할 수 있도록 돕습니다.

### 핵심 가치 (Key Values)
- **통합 관리**: 암호화폐와 주식을 아우르는 종합 자산 대시보드.
- **정확한 성과 분석**: 매수/매도 트랜잭션 기반의 정밀한 포지션 및 수익률(PnL) 계산.
- **개인화 경험**: 사용자별 맞춤 자산 등록 및 관심 종목 알림 설정.
- **자동화 확장성**: 향후 자동 매매(Trading Bot) 및 AI 기반 분석 기능 탑재 예정.

## 2. 주요 기능 (Core Features)

### 2.1 대시보드 (Dashboard)
로그인 후 처음 마주하는 화면으로, 전체 자산의 현황을 한눈에 파악할 수 있습니다.
- **총 자산 가치**: 현재 보유한 모든 포트폴리오의 평가액 합계.
- **자산 구성 그래프**: 원형 차트(Pie Chart)를 통해 자산별 비중 시각화.
- **주요 지표**: 전일 대비 등락률, 총 평가 손익 확인.

### 2.2 포트폴리오 관리 (Portfolio Management)
개별 자산의 보유 현황을 상세히 관리합니다.
- **자산 목록**: 보유 중인 자산 리스트, 현재가, 보유 수량, 평가 금액, 수익률 표시.
- **포지션 추가**: 새로운 자산을 매수했거나 기존 보유분을 등록할 때 사용.
  - 티커(Symbol), 수량, 평단가를 입력하여 등록.
- **상세 보기**: 특정 자산을 클릭하여 거래 이력 및 상세 정보를 확인.

### 2.3 자산 탐색 및 등록 (Asset Discovery)
- **전역 자산(Global Assets)**: 비트코인(BTC), 애플(AAPL) 등 시스템에 미리 등록된 주요 자산 검색 및 추가.
- **커스텀 자산(Custom Assets)**: 시스템에 없는 자산을 사용자가 직접 등록하여 관리 가능 (나만의 자산).

### 2.4 사용자 인증 및 보안 (Authentication & Security)
- **회원가입/로그인**: 이메일 기반의 안전한 계정 생성 및 로그인.
- **보안**: Argon2 해싱 및 JWT 토큰 기반의 안전한 세션 관리.
- **데이터 분리**: 멀티 테넌시(Multi-tenancy) 아키텍처로 사용자 데이터의 철저한 격리 및 보호.

### 2.5 알림 서비스 (Notifications)
- **시세 알림**: 관심 자산이 목표 가격에 도달하거나 급변동 발생 시 이메일 알림 수신.
- **포트폴리오 리포트**: 每日(Daily) 자산 변동 내역 리포트 기능 (예정).

## 3. 사용자 가이드 (User Guide)

### 3.1 시작하기
1. **계정 생성**: 메인 화면의 [로그인/회원가입] 버튼을 클릭하여 이메일과 비밀번호로 가입합니다.
2. **로그인**: 가입한 계정으로 로그인하면 대시보드로 이동합니다.

### 3.2 자산 등록하기
1. 상단 메뉴의 **[자산]** 또는 **[포트폴리오]** 탭으로 이동합니다.
2. **[+ 자산 추가]** 버튼을 클릭합니다.
3. 자산의 심볼(예: BTC)을 검색하거나 직접 정보를 입력합니다.
4. 보유 수량과 평균 매수 단가를 입력하고 저장합니다.

### 3.3 포트폴리오 확인하기
- 대시보드에서 전체 흐름을 파악하고, 포트폴리오 메뉴에서 각 자산별 상세 성과를 모니터링하세요.
- 데이터는 실시간(또는 주기적)으로 업데이트되는 시장 가격을 반영합니다.
